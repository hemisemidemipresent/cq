Calculation Functions

q!income startround endround

shows how much money you will get if you pop all the bloons from startround to endround

Arguments:

Name Type Description
startround number The starting round to calculate the cash
endround number The ending round to calculate the cash
gamemode string (optional) The gamemode to calculate for (no spaces)

Example:

q!income 2 37 abr

q!cash cashneeded startround

shows which round you will get cashneeded from startround

Arguments:

Name Type Description
cashneeded number The cash needed
startround number The starting round

Example:

q!cash 20000 37

q!health blimp round

shows which round you will get cashneeded from startround

Arguments:

Name Type Description
bloon string The blimp to calculate the health for
round number The round where the blimp spawns

Example:

q!health MOAB 420

q!speed bloon round

shows the speed of bloon in round round

Arguments:

Name Type Description
bloon string The bloon/blimp to calculate the speed for
round number The round where the bloon/blimp spawns

Example:

q!speed MOAB 420

q!herolevel hero startround mapdifficultyID

NOTE: all the arguments are OPTIONAL, so you can just type q!herolevel1, and there will be an automated response

shows which round a specific hero will reach a particular level

Arguments:

Name Type Description
bloon string (optional) the hero name
startround number (optional) The round when the hero is placed
mapdifficultyID number (optional)
    beginner - 1
    intermediate - 2
    advanced - 3
    expert - 4

Example:

q!herolevel, it is advised to use the easier interface, use the advanced argument-based one if you are experienced

q!monkeyopolis totalfarmcost numberoffarms

shows money produced in a round, cost of upgrade and how many rounds it takes to break even

Arguments:

Name Type Description
totalfarmcost number the cost of all the farms
numberoffarms number the number of farms sacrificed to the monkeyopolis

Example:

q!monkeyopolis 66000 3
