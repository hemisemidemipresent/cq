Tower Data Function

q!towername totalfarmcost numberoffarms

shows money produced in a round, cost of upgrade and how many rounds it takes to break even

Name Type Description
towername string the name of the tower, NO SPACES ALLOWED

Arguments:

Name Type Description
path number the upgrade path for the tower, for example: 030.
No crosspaths like 130, 032

Example:

q!monkeyopolis 66000 3
