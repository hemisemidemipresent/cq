Welcome to the Cyber Quincy Documentation!

Cyber Quincy is a unofficial Discord bot made to show information about Bloons TD 6, a Ninja Kiwi game.

more specifically, it is to have a vast array of game knowledge all condensed into one bot for easy reference and retrieval of data inside discord without needing external resources

Invite to your server!

Discord server
Required <> means that you must use this argument for the command to work
Optional [] means that you dont have to use this argument for the command to work

WARNING: Do not literally type out [], <>, =, etc...
