Calculation Functions

q!income startround endround

shows how much money you will get if you pop all the bloons from startround to endround

Arguments:

Name Type Description
startround number The starting round to calculate the cash
endround number The ending round to calculate the cash
gamemode string (optional) The gamemode to calculate for (no spaces)

Example:

q!income 2 37 abr

q!income startround endround

shows which round you will get cashneeded from startround

Arguments:

Name Type Description
cashneeded number The cash needed
startround number The starting round

Example:

q!cash 20000 37
